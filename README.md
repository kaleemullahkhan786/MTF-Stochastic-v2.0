# MTF Stochastic v2.0

## Overview

**_MTF Stochastic v2.0** is an incredibly powerful Multi-Timeframe upgrade to the classic Stochastic Oscillator. It allows you to visualize the Stochastic of *any higher timeframe* directly on your current trading chart. By utilizing a bespoke interpolation algorithm, it renders entirely smooth indicator lines eliminating the severe aesthetic "step" patterns traditionally seen in MTF tools.

## Key Features

- **True Multi-Timeframe (MTF) Display:** Effortlessly track the H4 or Daily Stochastic momentum while executing entries on your M15 chart.
- **Smooth Interpolation Engine:** Blends the values between higher timeframe candles to draw a fluid, professional curved line instead of blocky stairs.
- **Live Current Candle Projection:** Intelligently plots the unclosed live segment of the higher timeframe candle right on the edge of your chart for leading cues.
- **On-Chart Countdown Clock:** Includes an elegant, discrete text counter updating the exact time remaining until the current candle officially closes.

## Supported Platforms

This repository provides full compatibility with both major MetaTrader platforms. Standardizing your trading arsenal has never been easier! Inside the repository, you will find both the source code and the compiled trading files:

- **MetaTrader 4 (MT4):** `.mq4` (Source Code) and `.ex4` (Compiled Executable)
- **MetaTrader 5 (MT5):** `.mq5` (Source Code) and `.ex5` (Compiled Executable)

Whether you are an MT4 loyalist or an MT5 adapter, this indicator is ready to be deployed. The compiled files are included so you can start trading right away without needing to compile the code yourself.

## How to Install

1. Download the compiled files (`.ex4` for MT4 or `.ex5` for MT5) from this repository.
2. Open your MetaTrader platform.
3. Click on `File` -> `Open Data Folder` in the top menu.
4. For MT4, navigate to the `MQL4/Indicators` folder. For MT5, navigate to the `MQL5/Indicators` folder.
5. Paste the downloaded files into the folder.
6. Refresh the "Navigator" panel in MetaTrader, or simply restart the platform.
7. Find the indicator in the Navigator, drag and drop it onto your desired chart, and start trading!
